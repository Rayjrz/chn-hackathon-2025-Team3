# CHN Hackaton
This project is for a [hackaton](https://escience.washington.edu/events/brain-charts-hackathon/) that took place Nov 3-4 at the University of Washington eScience Institute, exploring the [Reproducible Brain Charts (RBC) dataset](https://reprobrainchart.github.io/)

The main/template repo for the hackathon is [here](https://github.com/uwescience/chn-hackathon-2025). It contains more information about the specific task, which is to predict the amount of psychopathology "p-factor" from neuroimaging data from the Philadelphia Neurodevelopment Cohort (PNC) within the RBC
